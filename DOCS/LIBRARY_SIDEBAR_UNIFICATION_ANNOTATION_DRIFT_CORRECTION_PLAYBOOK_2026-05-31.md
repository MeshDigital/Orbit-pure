# Library Sidebar Unification - Annotation Drift Correction Playbook (2026-05-31)

## Purpose

Provide correction steps for annotation drift discovered in quick-start index entries.

## Playbook

1. Detect drift class (duplicate intent, vague scope, verbose phrasing).
2. Normalize to outcome-first one-line annotation.
3. Align terms with status and recap language.
4. Verify neighboring entries remain distinct.

## Quality Gate

Correction is accepted only when scan readability improves without losing specificity.
