# Library Sidebar Unification Cross-Link Minimalism (2026-05-29)

Status: Cross-link minimalism guide
Date: 2026-05-29
Scope: Rules for keeping closure-doc cross-linking useful without creating noisy duplication.

## Rules

1. Link from the documentation index to the active artifact set.
2. Link between dated artifacts only when the relationship is operationally important.
3. Avoid linking every artifact to every other artifact.
4. Let the architecture contract define the minimum required discoverability surface.
