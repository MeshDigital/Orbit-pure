# Library Sidebar Unification - Role Continuity Incident Escalation Map (2026-05-31)

## Purpose

Map escalation paths for continuity incidents that can disrupt wave completion.

## Escalation Map

- Local correction path for low-impact incidents
- Contract-alignment escalation for assertion/index mismatches
- Governance escalation for repeated continuity failures

## Action Requirement

Every escalated incident must include owner, scope, and resolution checkpoint.
