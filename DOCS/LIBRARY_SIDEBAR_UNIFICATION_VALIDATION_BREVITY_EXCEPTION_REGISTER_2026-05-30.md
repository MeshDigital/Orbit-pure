# Library Sidebar Unification - Validation Brevity Exception Register (2026-05-30)

## Purpose

Track approved exceptions to brevity standards in validation reporting.

## Register Entries

- Exception reason
- Expanded detail required
- Impacted recap surface
- Approval wave and reviewer
- Sunset condition for returning to normal brevity

## Rule

Exceptions are temporary and must be cleared by the next wave unless re-approved.
