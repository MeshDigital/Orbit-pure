# Library Sidebar Unification Supersession Protocol (2026-05-29)

Status: Supersession protocol draft
Date: 2026-05-29
Scope: Rules for replacing or extending dated closure artifacts.

## Protocol

1. Do not overwrite historical closure docs to reflect new state; create a new dated artifact.
2. Update `DOCUMENTATION_INDEX.md`, `DOCUMENTATION_STATUS.md`, and the architecture doc-index contract test in the same change.
3. Keep prior dated artifacts available for auditability unless there is a deliberate archival policy change.
4. Note the superseding relationship in the new artifact rather than rewriting the old one.
