Status: Active style guide

Scope: Define lightweight lint-style shorthand for annotation consistency checks.

## Lint Shorthand

- `P1`: Purpose-first wording present.
- `S1`: Status term matches approved set.
- `L1`: Length remains short and scannable.
- `N1`: Stable noun used for repeated concept.
- `D1`: No duplicated near-synonym sequence.

## Example

`P1 S1 L1 N1 D1` means the annotation passed all shorthand lint checks.
