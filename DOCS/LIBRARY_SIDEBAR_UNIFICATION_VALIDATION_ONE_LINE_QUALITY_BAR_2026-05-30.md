# Library Sidebar Unification Validation One-Line Quality Bar (2026-05-30)

Status: Active quality bar
Date: 2026-05-30
Scope: Define minimum quality for one-line validation statements in recap surfaces.

## Quality Bar

1. Include numeric result.
2. Name the scope (focused gate/suite family) once.
3. Include baseline-change note only when it changed.
4. Avoid repeating the same one-line text across multiple sections.

## Good Example

Focused gate remained green (42 passed, 0 failed); suite scope unchanged; warning baseline unchanged.
