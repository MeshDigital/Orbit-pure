# Library Sidebar Unification - Role Continuity Incident Response Card (2026-05-31)

## Purpose

Provide a fast-response card for continuity incidents that threaten wave completion integrity.

1. Identify incident scope and impacted surfaces.
2. Apply immediate correction to highest-risk surface first.
3. Synchronize remaining surfaces in same execution window.

5. Record response outcome in memory tactics.

## Completion
