# Library Sidebar Unification Annotation Drift Examples (2026-05-29)

Status: Active reference
Date: 2026-05-29
Scope: Provide examples of annotation drift and practical normalization direction.

## Drift Examples

1. Mixed nouns for the same intent (guide, playbook, helper) in adjacent entries.
2. Process-heavy annotations that hide artifact purpose.
3. Overly long descriptions that reduce quick-start scan speed.
4. Inconsistent status words for equivalent artifacts.

## Normalization

Use one purpose noun per artifact, keep wording short, and align status terms across surfaces.
