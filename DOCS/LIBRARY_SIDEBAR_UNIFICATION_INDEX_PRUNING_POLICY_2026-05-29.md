# Library Sidebar Unification Index Pruning Policy (2026-05-29)

Status: Documentation index pruning policy draft
Date: 2026-05-29
Scope: Rules for trimming or regrouping quick-start references without breaking closure discoverability.

## Policy

1. Do not remove an active closure artifact from the index unless it is clearly superseded.
2. If an artifact is superseded, update the architecture doc-index contract in the same change.
3. Prefer regrouping or relabeling over silent removal.
4. Keep the quick-start section readable, but not at the cost of discoverability.
