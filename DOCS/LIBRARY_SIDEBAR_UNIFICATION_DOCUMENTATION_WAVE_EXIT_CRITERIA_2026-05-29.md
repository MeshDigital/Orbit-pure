# Library Sidebar Unification Documentation Wave Exit Criteria (2026-05-29)

Status: Active rubric
Date: 2026-05-29
Scope: Define when a documentation-only closure-maintenance wave is complete enough to close.

## Exit Criteria

1. All planned artifacts for the batch exist.
2. The documentation-index contract includes the full batch.
3. DOCUMENTATION_INDEX.md and DOCUMENTATION_STATUS.md describe the new family.
4. RECENT_CHANGES.md and both plan docs record the batch and the next queue.
5. Repo memory captures the reusable tactic and not the whole recap.
6. The focused sidebar-unification gate passes.

## Non-Exit Signals

A wave is not complete if any artifact is unlinked, any recap surface is stale, or the gate result has not been recorded.
