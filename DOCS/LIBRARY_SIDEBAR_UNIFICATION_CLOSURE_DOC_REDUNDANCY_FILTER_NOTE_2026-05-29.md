# Library Sidebar Unification Closure-Doc Redundancy Filter Note (2026-05-29)

Status: Active guidance
Date: 2026-05-29
Scope: Reduce archive bloat by filtering duplicate closure-maintenance narrative.

## Filter Rules

1. If two docs answer the same question, keep the clearer one and supersede the other.
2. Keep checklists procedural and keep notes explanatory; do not mix both heavily.
3. Prefer extending an existing artifact over creating a near-duplicate title.
4. Archive historical context only when it improves triage or onboarding.

## Output

The archive stays dense but intentional, with less overlap and better scan speed.
