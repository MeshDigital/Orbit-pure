# Library Sidebar Unification - Annotation Clarity Benchmark Pack (2026-05-30)

## Purpose

Standardize one-line index annotations so maintainers can scan large artifact sets quickly.

## Benchmark Rules

- One sentence per artifact annotation
- Lead with outcome, then scope qualifier
- Avoid duplicate phrasing across adjacent entries
- Keep terminology aligned with status and recap wording

## Pass/Fail Heuristic

- Pass: annotation remains precise without opening the target doc
- Fail: annotation repeats nearby wording or hides intent

## Operational Use

Use this benchmark before adding each new 10-item block to DOCUMENTATION_INDEX.
